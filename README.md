# Advent of Code – Entrega intermedia

Este repositorio contiene las soluciones que he desarrollado hasta el momento para varios días del reto *Advent of Code*, como parte de la práctica de la asignatura.  
La entrega incluye los siguientes días resueltos:

- **Día 2 – Gift Shop**  
- **Día 4 – Printing Department**  
- **Día 5 – Cafeteria**  
- **Día 7 – Laboratories**  
- **Día 8 – Playground**

Cada día tiene su propia carpeta y un archivo README interno donde explico:

1. Por qué he elegido ese problema.  
2. Qué técnica o estructura de datos de clase he utilizado.  
3. Cómo he abordado y resuelto la solución.  
4. Alternativas consideradas o descartadas.  
5. Valoración personal de lo aprendido.

Además, cada carpeta incluye:
- Código fuente (`.c` o `.cpp`)
- Archivos de entrada necesarios (`input.txt`)
- README específico del día

Este repositorio corresponde a la entrega intermedia solicitada para revisión del progreso del trabajo.
